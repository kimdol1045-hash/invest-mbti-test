## 1. 앱 개요

### 앱 이름
- **한글**: 퇴사준비 완료
- **영문**: QuitReady

### 한줄 소개
> "퇴직금부터 실업급여, 4대보험 정산까지 -- 퇴사에 필요한 모든 것을 한 곳에서 확인하세요."

### 타겟 사용자
| 구분 | 상세 |
|------|------|
| **핵심 타겟** | 퇴사/이직을 준비 중인 20~40대 직장인 |
| **2차 타겟** | 이미 퇴사한 후 실업급여/4대보험 처리를 해야 하는 구직자 |
| **3차 타겟** | 이직을 고민 중이라 퇴직금/실업급여를 미리 시뮬레이션해보고 싶은 직장인 |

### 사용자 페인포인트
1. 퇴직금이 정확히 얼마인지 모르겠어요 (평균임금, 통상임금 차이)
2. 실업급여를 받을 수 있는지, 얼마나 받을 수 있는지 모르겠어요
3. 4대보험 상실신고, 임의계속가입 등 뭘 해야 하는지 모르겠어요
4. 퇴사 전후로 해야 할 일들의 타임라인이 복잡해요
5. 관련 정보가 고용노동부, 국민연금공단, 건보공단 등에 분산되어 있어요

## 2. 핵심 기능

### MVP 1차 (2주 개발)

#### 2-1. 퇴직금 계산기
사용자가 입력한 정보를 기반으로 예상 퇴직금을 계산하는 핵심 기능.

**입력 항목:**
- 입사일 / 퇴사(예정)일
- 최근 3개월 월급여 (세전)
- 최근 3개월 상여금 총액 (있는 경우)
- 연차수당 (미사용 연차 x 일급)

**계산 공식 (근로기준법 제2조, 근로자퇴직급여 보장법 제8조):**
```
1일 평균임금 = 퇴직일 이전 3개월간 임금 총액 / 퇴직일 이전 3개월간 총 일수
퇴직금 = 1일 평균임금 x 30일 x (총 재직일수 / 365일)
```

**주요 로직:**
- 3개월간 임금 총액 = 기본급 + 상여금(3개월분 비례) + 연차수당(비례) + 기타 수당
- 통상임금 vs 평균임금 비교 후 높은 금액 적용 (근로기준법 제2조제2항)
- 2024년 대법원 전원합의체 판결 반영: 통상임금 산정 시 '고정성' 요건 폐지로 성과급/정기상여금 포함 가능

**지급 기준:**
- 계속근로기간 1년 이상
- 4주 평균 1주 소정근로시간 15시간 이상
- 퇴사일로부터 14일 이내 지급 의무 (근로기준법 제36조)

**결과 화면:**
- 예상 퇴직금 금액 (세전)
- 퇴직소득세 예상 공제액
- 실수령 퇴직금 예상액
- 면책 문구 표시

#### 2-2. 실업급여 자격 진단
사용자의 퇴사 사유와 고용보험 가입 기간을 기반으로 실업급여 수급 가능 여부와 예상 금액을 안내.

**수급 자격 조건 (고용보험법 제40조):**
1. 이직일 이전 18개월간 피보험 단위기간 통산 180일 이상 (초단시간근로자: 24개월간 180일)
2. 비자발적 이직 (권고사직, 계약만료, 정당한 사유의 자발적 이직 포함)
3. 근로의 의사와 능력이 있으나 취업하지 못한 상태
4. 재취업을 위한 적극적인 노력 의사

**수급 제한 사유:**
- 자기 사정으로 이직 (단순 전직, 개인 사유)
- 중대한 귀책사유로 해고 (형법/직무 관련 법률 위반 등)

**소정급여일수 (고용보험법 제50조):**

| 연령/피보험기간 | 1년 미만 | 1~3년 | 3~5년 | 5~10년 | 10년 이상 |
|----------------|---------|-------|-------|--------|----------|
| **50세 미만** | 120일 | 150일 | 180일 | 210일 | 240일 |
| **50세 이상 및 장애인** | 120일 | 180일 | 210일 | 240일 | 270일 |

**급여 계산 (2026년 기준):**
```
1일 구직급여 = 이직 전 평균임금의 60%
상한액: 1일 68,100원 (2026년 기준)
하한액: 1일 66,048원 (= 최저임금 10,320원 x 8시간 x 80%)
```

**입력 항목:**
- 퇴사 사유 (선택형: 권고사직/계약만료/자발적 퇴사/해고 등)
- 고용보험 가입 기간
- 나이
- 최근 3개월 평균 월급여

**결과 화면:**
- 수급 자격 여부 (O/X)
- 수급 불가 시 사유 안내
- 예상 1일 수급액
- 예상 총 수급기간 (소정급여일수)
- 예상 총 수급 금액
- 실업급여 신청 절차 안내 (고용센터 방문 -> 수급자격 인정 -> 구직활동 -> 실업인정)

#### 2-3. 4대보험 정산 가이드
퇴사 시 4대보험(국민연금, 건강보험, 고용보험, 산재보험) 처리 절차를 타임라인 형태로 안내.

**보험별 정산 안내:**

| 보험 | 상실신고 기한 | 퇴사자 조치 사항 |
|------|-------------|----------------|
| **국민연금** | 퇴사일 다음달 15일까지 | 임의계속가입 또는 지역가입자 전환 선택 |
| **건강보험** | 퇴사일로부터 14일 이내 | 지역가입자 전환 / 피부양자 등록 / 임의계속가입 중 선택 |
| **고용보험** | 퇴사일 다음달 15일까지 | 실업급여 신청 시 자동 처리 |
| **산재보험** | 퇴사일 다음달 15일까지 | 사업주 전액 부담, 근로자 별도 조치 불필요 |

**건강보험 상세:**
- 퇴직 정산: 해당 연도 실제 보수를 기반으로 보험료 재산정 (추가 납부 또는 환급 발생 가능)
- 임의계속가입: 퇴사 후 36개월간 직장가입자 보험료 유지 가능 (퇴직일로부터 60일 이내 신청)
- 피부양자 등록: 소득/재산 요건 충족 시 가족의 피부양자로 등록 (보험료 0원)

**국민연금 상세:**
- 퇴사일이 1일인 경우: 해당 월 보험료 미부과
- 퇴사일이 2일 이후인 경우: 해당 월 1개월분 보험료 부과
- 실업 크레딧: 구직급여 수급 중 국민연금 보험료의 75%를 국가가 지원 (최대 12개월)

#### 2-4. 퇴사 타임라인 체크리스트
퇴사 전후 해야 할 일들을 시간순으로 정리한 인터랙티브 체크리스트.

**타임라인 구성:**

```
[퇴사 4주 전]
 - 퇴직 의사 전달 (민법 제660조: 해지통고 후 1개월 경과 시 효력)
 - 잔여 연차 확인 및 사용 계획

[퇴사 2주 전]
 - 업무 인수인계 문서 작성
 - 회사 자산 반납 목록 확인 (노트북, 사원증, 법인카드 등)
 - 경력증명서/재직증명서 발급 요청

[퇴사 1주 전]
 - 개인 데이터 백업
 - 퇴직금 지급 일정 확인
 - 건강보험 임의계속가입 여부 결정

[퇴사 당일]
 - 사원증/장비 반납
 - 퇴직금/마지막 급여 정산 확인
 - 이직확인서 발급 요청 (실업급여용)

[퇴사 후 1주 이내]
 - 고용센터 방문하여 실업급여 신청 (이직확인서 필요)
 - 건강보험 자격 변동 처리

[퇴사 후 2주 이내]
 - 퇴직금 입금 확인 (14일 이내 미지급 시 지연이자 연 20% 청구 가능)
 - 국민연금 자격 변동 확인

[퇴사 후 1개월]
 - 원천징수영수증 수령
 - 이직 시 새 직장 4대보험 취득 확인
```

### 확장 2차 (MVP 이후)

#### 2-5. 이직 시 연봉 비교 시뮬레이터
- 현재 연봉 vs 이직 후 연봉 실수령액 비교
- 4대보험료, 소득세 자동 계산
- 퇴직금 포기 비용 대비 이직 수익 분석

#### 2-6. 퇴직소득세 계산기
- 근속연수공제, 환산급여 공제 적용
- 퇴직소득세 = (환산산출세액 / 12 x 근속연수)
- 세후 실수령 퇴직금 상세 계산

#### 2-7. 실업급여 일정 관리
- 실업인정일 알림 (4주마다 고용센터 방문)
- 구직활동 기록 관리
- 수급 잔여일수/금액 추적

#### 2-8. AI 퇴사 상담 (생성형 AI)
- 퇴사 사유별 맞춤 조언
- 사용 시 AI 고지 의무 준수 (앱인토스 정책)
- 모든 AI 응답에 가시적 AI 생성 표시 필수

#### 2-9. 공유 기능
- 체크리스트 진행률 이미지 생성 및 공유
- 토스 앱 내 공유 API 활용

## 5. 수익화 전략

### 5-1. IAA (인앱광고) 전략

앱인토스의 인앱광고는 Google AdMob 정책을 준수하며, 토스 Ads와 연동된다.

| 광고 유형 | 노출 위치 | 빈도 | 예상 eCPM |
|-----------|----------|------|-----------|
| **배너 광고** | 홈 화면 하단 | 상시 노출 | $0.5~1.5 |
| **전면 광고** | 퇴직금 계산 결과 확인 후 | 계산 완료 시 1회 | $3~8 |
| **보상형 광고** | 실업급여 상세 분석 잠금 해제 | 사용자 선택 시 | $10~20 |

### 5-2. 광고 배치 원칙
- 계산 결과를 먼저 보여준 후 추가 상세 분석에 보상형 광고 적용 (다크패턴 금지 준수)
- 핵심 기능(기본 계산)은 광고 없이 이용 가능
- 전면 광고는 자연스러운 화면 전환 시점에만 노출
- 과도한 광고로 UX를 해치지 않도록 1세션당 전면 광고 최대 2회 제한

### 5-3. 수익 정산
- 앱인토스 콘솔 > 워크스페이스 > 정산 정보 등록 필요
- 검토 소요: 영업일 2~3일
- 성과 메뉴에서 일별 광고 노출 수, eCPM, 예상 수익 확인 가능 (오전 3시 10분 전일 성과 업데이트)
- 수수료: CBT 기간 15% (현재 0%)

### 5-4. 예상 수익 시뮬레이션 (월간)

| 지표 | 보수적 | 낙관적 |
|------|--------|--------|
| DAU | 1,000 | 5,000 |
| 배너 노출 | 1,000 | 5,000 |
| 전면 광고 노출 | 500 | 2,500 |
| 보상형 광고 시청 | 100 | 500 |
| **월 예상 수익** | **약 30~50만원** | **약 150~300만원** |
