## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| **앱 이름 (한글)** | 키즈존 |
| **앱 이름 (영문)** | KidsZone |
| **한줄 소개** | 수유실, 키즈카페, 놀이터, 아이 동반 식당을 한눈에 찾아보세요 |
| **카테고리** | 생활/육아 |
| **타겟 사용자** | 영유아 자녀(0~7세)를 둔 부모, 예비 부모, 조부모 등 돌봄 관계자 |

### 핵심 가치
- **통합 검색**: 흩어져 있는 아이 동반 시설 정보를 하나의 지도에서 한눈에 확인해요
- **실시간 정보**: 공공데이터 API와 사용자 제보를 결합해 최신 정보를 제공해요
- **간편한 경험**: 토스 앱 안에서 별도 설치 없이 바로 이용할 수 있어요

### 문제 정의
- 수유실, 키즈카페, 놀이터, 아이 동반 가능 식당 정보가 각각 다른 플랫폼에 분산되어 있어요
- 기존 지도 앱에서는 "아이와 함께 갈 수 있는 곳"이라는 기준으로 필터링이 어려워요
- 시설의 실제 운영 여부, 청결도, 편의시설 유무 등 부모에게 중요한 정보가 부족해요

## 2. 핵심 기능

### MVP 1차 (필수 기능)

| 기능 | 설명 |
|------|------|
| **통합 지도 뷰** | 현재 위치 기반으로 수유실, 키즈카페, 놀이터, 아이 동반 식당을 카테고리별 마커로 지도에 표시해요 |
| **카테고리 필터** | 수유실 / 키즈카페 / 놀이터 / 아이 동반 식당 중 원하는 카테고리만 선택해서 볼 수 있어요 |
| **시설 상세 정보** | 주소, 운영시간, 전화번호, 편의시설(기저귀 교환대, 주차장 등) 정보를 확인할 수 있어요 |
| **길찾기 연동** | 카카오맵/네이버지도 앱으로 바로 길찾기를 할 수 있어요 |
| **리스트 뷰** | 현재 위치에서 가까운 순으로 시설 목록을 볼 수 있어요 |
| **키워드 검색** | 지역명이나 시설명으로 직접 검색할 수 있어요 |

### 확장 2차 (후속 기능)

| 기능 | 설명 |
|------|------|
| **사용자 리뷰 & 평점** | 방문한 시설에 대해 별점과 한줄 리뷰를 남길 수 있어요 (OAuth 인증 후) |
| **사용자 제보** | 지도에 없는 새로운 시설을 제보하거나 기존 정보를 수정 요청할 수 있어요 |
| **즐겨찾기** | 자주 가는 장소를 저장하고 빠르게 접근할 수 있어요 |
| **주변 추천 코스** | 놀이터 + 근처 식당처럼 아이와 함께하는 코스를 추천받을 수 있어요 |
| **시설 사진 업로드** | 방문한 시설의 실제 사진을 공유할 수 있어요 |
| **혼잡도 표시** | 사용자 체크인 기반으로 현재 혼잡도를 확인할 수 있어요 |
| **푸시 알림** | 즐겨찾기한 시설 주변에 가면 알림을 받을 수 있어요 |

## 3. 화면 구성

### 3.1 화면 목록

#### (1) 홈 화면 (지도 뷰) - `intoss://kids-zone`
- **상단**: TDS Navigation 바 (앱 타이틀 "키즈존")
- **카테고리 탭**: 전체 / 수유실 / 키즈카페 / 놀이터 / 아이 동반 식당 (수평 스크롤 칩)
- **지도 영역**: 카카오맵 기반 지도, 카테고리별 컬러 마커 표시
  - 수유실: 핑크 마커
  - 키즈카페: 오렌지 마커
  - 놀이터: 그린 마커
  - 아이 동반 식당: 블루(#3182F6) 마커
- **현재 위치 버튼**: 내 위치로 돌아가기 FAB 버튼
- **하단 시트**: 드래그 가능한 바텀시트로 가까운 시설 리스트 미리보기 (TDS ListRow)
- **검색 아이콘**: 상단 우측, 검색 화면으로 이동

#### (2) 리스트 뷰 화면 - `intoss://kids-zone/list`
- **상단**: TDS Navigation 바 + 정렬 옵션 (거리순 / 평점순)
- **카테고리 필터**: 홈과 동일한 카테고리 칩
- **시설 목록**: TDS ListRow 컴포넌트로 구성
  - 시설명, 카테고리 뱃지, 거리, 주소, 운영시간
  - 썸네일 이미지 (2차)
- **무한 스크롤**: 20개씩 페이지네이션

#### (3) 시설 상세 화면 - `intoss://kids-zone/detail/:id`
- **상단**: TDS Navigation 바 (뒤로가기)
- **시설 기본 정보**: 시설명, 카테고리 뱃지, 주소, 전화번호
- **지도 미니뷰**: 해당 시설 위치를 작은 지도로 표시
- **편의시설 태그**: 기저귀 교환대, 유아 의자, 주차 가능, 수유 공간 등 아이콘 + 텍스트
- **운영 정보**: 운영시간, 이용료, 이용 안내
- **길찾기 버튼**: TDS BottomCTA 버튼 ("길찾기")으로 외부 지도 앱 연동
- **리뷰 섹션 (2차)**: 별점 평균, 리뷰 목록

#### (4) 검색 화면 - `intoss://kids-zone/search`
- **상단**: TDS 검색 입력 필드 (자동 포커스)
- **최근 검색어**: 최근 검색한 키워드 목록 (로컬 저장)
- **인기 검색어**: 많이 검색된 지역/시설 (2차)
- **검색 결과**: TDS ListRow로 검색 결과 표시

#### (5) 즐겨찾기 화면 (2차) - `intoss://kids-zone/favorites`
- **상단**: TDS Navigation 바 ("즐겨찾기")
- **즐겨찾기 목록**: TDS ListRow 형태, 스와이프 삭제 지원

### 3.2 화면 플로우

```
[홈 화면 (지도)]
    ├─ 카테고리 탭 선택 → 지도 마커 필터링
    ├─ 마커 클릭 → [시설 상세]
    ├─ 하단 시트 리스트 아이템 클릭 → [시설 상세]
    ├─ 리스트 뷰 전환 버튼 → [리스트 뷰]
    ├─ 검색 아이콘 → [검색 화면]
    └─ 즐겨찾기 아이콘 (2차) → [즐겨찾기]

[리스트 뷰]
    ├─ 아이템 클릭 → [시설 상세]
    └─ 지도 뷰 전환 → [홈 화면 (지도)]

[시설 상세]
    ├─ 길찾기 버튼 → 외부 지도 앱 (카카오맵/네이버지도)
    ├─ 전화 버튼 → 전화 앱 연동
    ├─ 즐겨찾기 버튼 (2차) → 즐겨찾기 추가/제거
    └─ 리뷰 작성 (2차) → 리뷰 작성 모달

[검색 화면]
    ├─ 키워드 입력 → 검색 결과 리스트
    └─ 결과 아이템 클릭 → [시설 상세]
```

## 4. 기술 스택

## 5. 수익화 전략

### 수익 모델: IAA (인앱광고)

키즈존은 인앱광고(IAA) 기반으로 수익화해요. 사용자 경험을 해치지 않으면서 자연스러운 광고 노출을 추구해요.

#### 광고 배치 전략

| 광고 유형 | 위치 | 설명 |
|-----------|------|------|
| **네이티브 광고** | 리스트 뷰 중간 (5번째 아이템마다) | 시설 목록 사이에 "추천" 라벨을 붙여 자연스럽게 노출해요 |
| **배너 광고** | 시설 상세 화면 하단 | 길찾기 버튼 위에 작은 배너를 표시해요 |
| **스폰서 마커** | 지도 위 강조 마커 | 키즈카페/식당이 유료로 마커를 강조 표시할 수 있어요 |

#### 광고 타겟 카테고리 (기대 광고주)
- 키즈카페, 실내 놀이터 (지역 업체)
- 아이 동반 가능 패밀리 레스토랑
- 유아용품 브랜드 (기저귀, 분유, 유모차 등)
- 육아 교육 서비스
- 유아 보험 상품 (정보성 광고, 상품 직접 중개는 금지)

#### 수익 예상 (월 기준)

| 단계 | MAU | 예상 월 수익 |
|------|-----|-------------|
| 출시 초기 (1-3개월) | 5,000~10,000 | 50만~100만원 |
| 성장기 (4-6개월) | 30,000~50,000 | 200만~500만원 |
| 안정기 (7-12개월) | 100,000+ | 500만~1,000만원 |

#### 광고 정책 준수 사항
- 광고와 콘텐츠를 명확히 구분해요 ("광고", "추천" 라벨 필수)
- 다크패턴 금지: 실수로 클릭을 유도하는 배치 금지
- 전면 광고(인터스티셜)는 사용하지 않아요 (UX 저해)
- 아동 대상 부적절한 광고 차단 필터 적용

## 6. 앱인토스 정책 체크

### 6.1 위반 가능 항목 점검

| 정책 항목 | 상태 | 설명 |
|-----------|------|------|
| 가상자산/NFT | 해당없음 | 가상자산 관련 기능 없음 |
| 금융상품 중개 | 해당없음 | 금융상품 판매/중개 없음 |
| 사행성/복권 | 해당없음 | 해당 기능 없음 |
| 데이팅/소셜 | 해당없음 | 소셜 기능 없음 (리뷰는 1:N 정보 공유) |
| 외부 앱 설치 유도 | **주의 필요** | 길찾기 시 카카오맵/네이버지도 앱 실행 - 앱 "설치" 유도가 아닌 기존 설치 앱 "실행"으로 처리해야 해요. 미설치 시 웹 브라우저 길찾기로 폴백해요 |
| 외부 결제 페이지 | 해당없음 | 결제 기능 없음 (IAA 수익 모델) |

## 8. 경쟁력 & 차별점

### 8.1 기존 서비스와의 비교

| 비교 항목 | 네이버지도 / 카카오맵 | 맘카페 (카페 게시글) | **키즈존** |
|-----------|----------------------|---------------------|-----------|
| 수유실 검색 | 제한적 (카테고리 없음) | 게시글 검색 (비정형) | **전용 카테고리로 즉시 검색** |
| 키즈카페 검색 | 키워드 검색 가능 | 후기 중심 | **지도 마커로 한눈에 확인** |
| 놀이터 검색 | 일부 표시 | 정보 부족 | **공공데이터 기반 전수 표시** |
| 아이 동반 식당 | 구분 불가 | 추천 게시글 | **전용 필터로 쉽게 탐색** |
| 통합 검색 | 각각 별도 검색 | 불가 | **4개 카테고리 통합 지도** |
| 편의시설 정보 | 기본 정보만 | 후기에 산재 | **기저귀 교환대, 유아 의자 등 태그화** |
| 접근성 | 별도 앱 필요 | 카페 앱 필요 | **토스 앱 내 바로 이용** |

### 8.2 핵심 차별점

1. **"아이와 함께" 관점의 통합 지도**
   - 기존 지도 앱은 일반 사용자 대상이에요. 키즈존은 "아이와 외출하는 부모"라는 명확한 페르소나에 집중해요
   - 수유실, 키즈카페, 놀이터, 아이 동반 식당을 하나의 지도에서 동시에 볼 수 있는 유일한 서비스예요

2. **공공데이터 기반 신뢰성**
   - 행정안전부, 여성가족부 공공데이터를 기반으로 정확한 위치 정보를 제공해요
   - 사용자 제보로 데이터를 지속적으로 보강해요

3. **토스 앱 내 즉시 접근**
   - 3,000만 토스 사용자가 별도 설치 없이 바로 이용할 수 있어요
   - 외출 중 급하게 수유실을 찾아야 할 때 토스 앱만 열면 돼요

4. **육아 친화 UX**
   - 한 손으로 아이를 안고 조작할 수 있도록 하단 중심 인터랙션을 설계해요
   - 최소 탭으로 원하는 시설을 찾을 수 있어요 (홈 → 마커 클릭 → 길찾기, 3탭 완료)

5. **경쟁 진입장벽**
   - 공공데이터 수집 + 사용자 제보 데이터가 축적될수록 데이터 경쟁력이 강화돼요
   - 토스 플랫폼 내 육아 카테고리 선점 효과가 있어요

### 8.3 성장 전략

| 단계 | 기간 | 목표 |
|------|------|------|
| **시드** | 1-2개월 | MVP 출시, 서울/수도권 데이터 집중 |
| **성장** | 3-6개월 | 전국 확대, 사용자 리뷰 기능 오픈, MAU 5만 달성 |
| **확장** | 7-12개월 | 추천 코스, 혼잡도, 스폰서 마커 수익화 강화, MAU 10만 달성 |
| **고도화** | 12개월+ | AI 기반 맞춤 추천, 육아 커뮤니티 기능 검토 |
