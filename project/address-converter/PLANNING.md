> 작성일: 2026-03-06
> 플랫폼: 앱인토스 (Apps in Toss) WebView 미니앱
> 상태: MVP 기획

## 1. 앱 개요

### 앱 이름
- **한글**: 주소변환기
- **영문**: Address Converter

### 한줄 소개
한국 주소를 영문 및 세계 각국 언어로 즉시 변환하고, 해외 주소를 한국어로 이해할 수 있는 글로벌 주소 변환 서비스예요.

### 타겟 사용자
| 사용자 그룹 | 니즈 | 예상 비율 |
|-------------|------|-----------|
| 해외 직구 이용자 | 한국 주소를 영문으로 변환해서 배송지 입력 | 45% |
| 해외 서비스 가입자 | 해외 사이트 회원가입 시 영문 주소 필요 | 25% |
| 외국인 거주자/관광객 | 한국 주소를 모국어로 이해하고 싶을 때 | 15% |
| 해외 주소 이해 필요자 | 해외에서 받은 주소를 한국어로 파악할 때 | 10% |
| 유학생/해외 거주자 | 각종 서류에 영문/현지어 주소 기입 | 5% |

### 핵심 가치
- **빠른 변환**: 주소 입력 즉시 다국어 변환 결과 제공
- **정확성**: 정부 공공 API 기반 공식 영문 주소 표기
- **편의성**: 클립보드 복사 한 번으로 바로 사용 가능
- **다국어 지원**: 영어 외 일본어, 중국어 등 10개+ 언어 지원

## 2. 핵심 기능

### MVP 1차 (2주 개발)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **한국 주소 검색** | 도로명/지번 주소 자동완성 검색 | P0 |
| **영문 주소 변환** | 한국 주소를 영문 도로명 주소로 변환 | P0 |

## 3. 화면 구성

### 3-1. 화면 목록

#### [S1] 메인 화면 - 주소 변환
- **Navigation**: "주소변환기" 타이틀
- **SearchField**: 한국 주소 검색 입력창 (placeholder: "도로명 또는 지번 주소를 입력해 주세요")
- **Tab**: [영문 변환 | 다국어 변환 | 해외 -> 한글] (MVP에서는 영문/다국어만)
- **최근 변환 섹션**: ListRow 컴포넌트로 최근 변환 주소 목록
- **BottomCTA**: 없음 (검색 결과 화면에서 CTA 제공)
- **광고 영역**: 하단 배너 광고

#### [S2] 주소 검색 결과 화면
- **Navigation**: 뒤로가기 + "주소 검색"
- **SearchField**: 검색어 입력 상태 유지
- **검색 결과 리스트**: ListRow 컴포넌트로 검색 결과 표시
  - 좌측: 도로명주소 아이콘
  - 메인 텍스트: 도로명 주소
  - 서브 텍스트: 지번 주소, 우편번호
- **빈 상태**: 검색 결과가 없을 때 안내 메시지

#### [S3] 영문 변환 결과 화면
- **Navigation**: 뒤로가기 + "영문 주소"
- **원본 주소 섹션**:
  - ListHeader: "원본 주소"
  - 한글 도로명 주소 + 우편번호
- **영문 주소 섹션**:
  - ListHeader: "영문 주소"
  - 전체 영문 주소 (큰 폰트, 복사 가능)
  - Button: "주소 복사하기" (Primary, 토스 블루)
- **필드별 분리 섹션**:
  - ListHeader: "해외 사이트 입력용"
  - ListRow: Address Line 1 (도로명 + 건물번호)
  - ListRow: Address Line 2 (상세주소 - 동/호수)
  - ListRow: City (시/군/구)
  - ListRow: State/Province (시/도)
  - ListRow: Postal Code (우편번호)
  - ListRow: Country (South Korea)
  - 각 행 우측에 복사 아이콘
- **BottomCTA**: "다른 언어로도 변환하기" 버튼
- **광고 영역**: 결과 하단 전면형 광고 (변환 완료 후 노출)

#### [S4] 다국어 변환 결과 화면
- **Navigation**: 뒤로가기 + "다국어 주소"
- **원본 주소 섹션**: 한글 주소 표시
- **언어 선택 섹션**:
  - Tab 또는 SegmentedControl: 언어 선택
  - 지원 언어: English, 日本語, 中文(简), 中文(繁), Francais, Deutsch, Espanol, العربية, Portugues, Italiano
- **변환 결과 섹션**:
  - 선택한 언어로 변환된 주소
  - Button: "주소 복사하기"
- **전체 언어 결과**:
  - ListRow로 각 언어별 변환 결과 나열
  - 각 행에 복사 버튼
- **광고 영역**: 다국어 변환 시 전면형 광고 (하루 최대 3회)

#### [S5] 변환 기록 화면
- **Navigation**: 뒤로가기 + "변환 기록"
- **기록 리스트**: ListRow 컴포넌트
  - 좌측: 언어 플래그 아이콘
  - 메인 텍스트: 원본 한글 주소 (축약)
  - 서브 텍스트: 변환 언어 + 변환 날짜
  - 우측: 복사 아이콘
- **빈 상태**: "아직 변환한 주소가 없어요"

### 4-2. 외부 API

#### (1) 도로명주소 API (juso.go.kr) - 한국 주소 영문 변환 [핵심]
- **제공**: 행정안전부
- **기능**: 한국 주소 검색 + 영문 주소 변환
- **엔드포인트**:
  - 한글 주소 검색: `https://business.juso.go.kr/addrlink/addrLinkApi.do`
  - 영문 주소 검색: `https://business.juso.go.kr/addrlink/addrEngApi.do`
- **비용**: 무료 (공공데이터)
- **한도**: 일 10,000건 (신청 시 증량 가능)
- **장점**: 정부 공식 데이터, 정확한 영문 표기법 준수, 무료
- **단점**: 영문만 지원 (다국어 미지원)
- **발급**: data.go.kr에서 API 키 발급

#### (2) Google Geocoding API - 다국어 주소 변환 [핵심]
- **제공**: Google Maps Platform
- **기능**: 주소 -> 좌표 변환, `language` 파라미터로 다국어 결과 반환
- **엔드포인트**: `https://maps.googleapis.com/maps/api/geocode/json`
- **비용**: 월 10,000건 무료 / 초과 시 건당 $0.005
- **한도**: 분당 3,000건
- **지원 언어**: 70개+ 언어 (ja, zh-CN, zh-TW, fr, de, es, ar, pt, it 등)
- **장점**: 전세계 주소 지원, 다국어 결과, 역지오코딩(좌표->주소) 가능
- **단점**: 무료 한도 초과 시 유료, 지역 주소명은 번역이 아닌 음역(transliteration) 제공

#### (3) Kakao 로컬 API - 한국 주소 검색 [보조]
- **제공**: Kakao Developers
- **기능**: 한국 주소 검색, 좌표 변환
- **엔드포인트**: `https://dapi.kakao.com/v2/local/search/address.json`
- **비용**: 무료 (일 100,000건)
- **장점**: 한국 주소에 특화, 넉넉한 무료 한도
- **단점**: 영문/다국어 변환 기능 없음
- **활용**: 한국 주소 자동완성 검색 보조용

#### (4) Google Cloud Translation API - 주소 번역 [보조]
- **제공**: Google Cloud
- **기능**: 텍스트 번역 (주소의 일반 명사 부분 번역 보완)
- **비용**: 월 500,000자 무료 / 초과 시 100만 자당 $20
- **활용**: Geocoding API의 음역 결과를 보완하는 용도

#### API 선정 전략 (MVP)
```
[한국 주소 검색] --> 도로명주소 API (무료, 공식)
        |
[영문 변환] --> 도로명주소 영문 API (무료, 공식 표기법)
        |
[다국어 변환] --> Google Geocoding API (language 파라미터)
        |
[보완 번역] --> Google Translation API (음역 보완, 2차에서 적용)
```

### 4-3. 데이터 저장

| 저장소 | 용도 | 방식 |
|--------|------|------|
| 네이티브 스토리지 | 최근 변환 기록 (최대 50건) | `@apps-in-toss/web-framework` 스토리지 API |

### 5-2. 광고 정책 준수

- Google AdMob 광고 정책 준수 (앱인토스 IAA 정책)

| 네이버 영문주소 | 포털 검색 | 접근성 좋음 | 영문만 지원, 필드 분리 없음 |

### 8-2. 주소변환기의 차별점

| 차별점 | 설명 |
|--------|------|
| **토스 앱 내 즉시 접근** | 별도 앱 설치/웹 접속 없이 토스에서 바로 사용 |
| **다국어 변환 (10개+ 언어)** | 기존 서비스는 대부분 영문만 지원, 본 서비스는 일본어/중국어/프랑스어 등 다국어 지원 |
| **해외 사이트 입력 형식 맞춤** | Address Line 1/2, City, State, Zip 등 필드별 분리 제공 |
| **원터치 복사** | 전체 주소 또는 필드별 개별 복사 기능 |
| **TDS 기반 깔끔한 UI** | 토스 디자인 시스템으로 일관되고 직관적인 사용 경험 |
| **변환 기록 관리** | 이전에 변환한 주소를 다시 찾아볼 수 있어 반복 사용에 편리 |
| **무료 서비스** | 광고 기반 수익으로 사용자에게 무료 제공 |

### 8-3. 핵심 사용 시나리오별 경쟁력

#### 시나리오 1: 해외 직구 배송지 입력
> "Amazon에서 주문하려는데 영문 주소를 어떻게 쓰지?"
- 기존: 네이버 검색 -> juso.go.kr 접속 -> 영문 주소 확인 -> 수동으로 필드 분리
- **주소변환기**: 토스에서 바로 검색 -> Address Line 1/2, City, State, Zip 자동 분리 -> 각 필드 복사

#### 시나리오 2: 해외 사이트 회원가입
> "해외 서비스에 가입하는데 주소를 영어로 써야 해요"
- 기존: 여러 사이트 비교하며 영문 표기법 확인
- **주소변환기**: 한 번 검색으로 공식 영문 주소 확인 + 복사

#### 시나리오 3: 외국인에게 주소 전달
> "일본 친구에게 한국 집 주소를 일본어로 보내고 싶어요"
- 기존: 해당 서비스 없음 (수동 번역 또는 영문 전달)
- **주소변환기**: 다국어 탭에서 일본어 선택 -> 변환된 주소 공유

#### 시나리오 4: 해외 주소 이해 (2차 기능)
> "해외에서 받은 주소가 무슨 뜻인지 모르겠어요"
- 기존: Google 번역기로 수동 번역 (부정확)
- **주소변환기**: 해외 주소 입력 -> 한국어 변환 + 지도 위치 확인

## 부록: API 상세 사양

### A. 도로명주소 API (juso.go.kr)

**한글 주소 검색 요청**
```
GET https://business.juso.go.kr/addrlink/addrLinkApi.do
?confmKey={API_KEY}
&currentPage=1
&countPerPage=10
&keyword={검색어}
&resultType=json
```

**영문 주소 검색 요청**
```
GET https://business.juso.go.kr/addrlink/addrEngApi.do
?confmKey={API_KEY}
&currentPage=1
&countPerPage=10
&keyword={검색어}
&resultType=json
```

**주요 응답 필드 (영문)**
| 필드 | 설명 | 예시 |
|------|------|------|
| roadAddr | 영문 도로명주소 | 35, Sejong-daero, Jung-gu, Seoul |
| jibunAddr | 영문 지번주소 | 211, Taepyeongno 1(il)-ga, Jung-gu, Seoul |
| zipNo | 우편번호 | 04524 |
| engSiNm | 시도 영문 | Seoul |
| engGguNm | 구 영문 | Jung-gu |
| engRoadNm | 도로명 영문 | Sejong-daero |
