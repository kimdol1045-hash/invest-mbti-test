# 투자, 나답게 - 앱인토스 콘솔 등록 가이드

## 1단계: 콘솔 접속 & 워크스페이스 생성

1. https://apps-in-toss.toss.im/ 접속
2. 토스 계정으로 로그인
3. 워크스페이스 생성 (아직 없는 경우)
   - 워크스페이스 = 앱을 관리하는 조직 단위
   - 사업자등록번호 필요 (개인 개발자도 가능한지 확인 필요)

## 3단계: 등록 후 발급받을 것들

### 바로 발급
- [ ] **앱 ID** → `granite.config.ts`의 `appName`에 입력
- [ ] **OAuth Client ID** → `src/utils/auth.ts`의 `AUTH_CONFIG.CLIENT_ID`에 입력
- [ ] **OAuth Redirect URI** 등록 → 앱 URL + `/auth/callback`

### mTLS 인증서 (서버 API 사용 시)
- [ ] 콘솔에서 인증서 발급 → 서버에 설치
- 현재는 서버 없이 클라이언트만 사용하므로 **당장은 불필요**

### AdMob 광고 (수익화 시)
- [ ] 콘솔에서 AdMob 키 연동 설정
- [ ] 교육 카테고리 수익 자격 확인 (콘솔 문의)

---

## 4단계: OAuth 연동 체크리스트

Client ID 발급 후 해야 할 것:

```
1. src/utils/auth.ts 열기
2. AUTH_CONFIG.CLIENT_ID에 발급받은 값 입력
3. 콘솔에서 Redirect URI 등록: https://{앱도메인}/auth/callback
4. 끝! 로그인 플로우 자동 동작
```

이미 구현된 기능:
- ✅ OAuth 인증 플로우 (authorize → callback → token)
- ✅ AuthProvider 전역 상태 관리
- ✅ x-toss-user-key 헤더 자동 주입
- ✅ 로그인/로그아웃 함수

## 주의사항

### 교육 카테고리 수익 관련
- 정책: "교육 카테고리에서 수익을 내려면 국가 자격증 필요"
