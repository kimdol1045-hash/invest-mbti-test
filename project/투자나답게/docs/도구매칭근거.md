# 투자 성향 유형별 관심 도구 매칭 근거

## 매칭 프레임워크

각 MBTI 축의 특성에 따라 도구의 성격을 분류하고, 4축 조합으로 6개 도구를 배정한다.

### 축별 도구 성향 분류

#### E/I (소통형/독립형)
- **E**: 대중적이고 널리 알려진 도구 (이동평균선, RSI, MACD, 볼린저밴드)
- **I**: 심층적이고 전문가 선호 도구 (일목균형표, 피보나치, ADX, ATR)

#### S/N (데이터형/직감형)
- **S**: 구체적 수치 기반 지표 (RSI, 스토캐스틱, 거래량이동평균, CCI, VWAP, ATR)
- **N**: 큰 흐름·패턴 기반 도구 (MACD, 일목균형표, 피보나치, OBV, 켈트너채널)

#### T/F (논리형/감성형)
- **T**: 규칙 명확, 객관적 신호 (RSI, ADX, 파라볼릭SAR, 장악형, 마루보즈)
- **F**: 시각적, 직관적 해석 (볼린저밴드, 이동평균선, 추세선, 망치형, 도지)

#### J/P (계획형/유연형)
- **J**: 기준선·확정적 패턴 (지지저항선, 더블탑/바텀, 헤드앤숄더, 이동평균선)
- **P**: 유연·적응적 패턴 (삼각수렴, 플래그페넌트, 쐐기형, 볼린저밴드)

### 배정 규칙
- 각 유형에 **보조지표 3개 + 캔들 패턴 1~2개 + 차트 패턴 1~2개 = 총 6개** 배정
- 4축 성향 조합에 따라 도구 풀에서 가장 적합한 항목 선정
- "이 성향이 **관심을 갖는 경향이 있대요**"라는 정보 제공 (추천 아님)

## 신뢰도 검증

### 내적 일관성 ✅
- 같은 축 성향을 공유하는 유형들은 유사한 도구 풀을 공유
  - ST 조합(ISTJ, ISTP, ESTP, ESTJ): RSI, 스토캐스틱 등 수치 기반 도구 공통
  - NF 조합(INFJ, INFP, ENFP, ENFJ): OBV, 추세선 등 흐름 기반 도구 공통
  - NT 조합(INTJ, INTP, ENTP, ENTJ): 피보나치, ADX 등 복합 분석 도구 공통

### 정책 안전성 ✅
- "관심이 많대요" 표현 → 추천이 아닌 경향 정보
- 모든 유형에서 모든 도감 콘텐츠 열람 가능 (잠금 없음)
- 특정 도구 사용을 유도하는 행동 지시 없음

### 도구 커버리지
- 37개 도구 중 35개 사용 (95% 커버)
- 미사용: 별형(별도 분류 어려움) → 도감에서 직접 학습 유도
