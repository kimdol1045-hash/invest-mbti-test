# 앱인토스 정책 & 심사 기준

## 서비스 오픈 프로세스 (5단계)

1. **시작** - 콘솔에서 앱 등록 (1-2일)
2. **디자인** - App Builder 또는 Figma로 UI 제작
3. **개발** - SDK 연동 및 구현
4. **심사** - 4단계 검수 (2-3 영업일)
5. **출시** - 콘솔에서 "출시" 클릭

**개발 소요 기간:**
- 게임: ~2-4주
- 비게임: ~2-3개월

---

## 심사 기준 (4단계 검수)

| 단계 | 내용 |
|------|------|
| 운영 심사 | 앱 정보, 노출 설정, 문서화 확인 |
| 기능 심사 | 토스 앱 내 오류 없는 동작 확인 |
| 디자인 심사 | 토스 UI 가이드라인 준수 (TDS 사용 필수) |
| 보안 심사 | 개인정보 보호 및 데이터 보안 |

게임은 **등급분류위원회(GRAC) 등급 인증** 또는 앱스토어 자체 분류 필요.

---

## 등록 불가 서비스 (절대 금지)

- 디지털/가상 자산 관련 (NFT 포함)
- 현금/자산 직접 교환/환전/환불 서비스
- 불법 행위 조장
- 사행성/복권/베팅
- 금융 상품 중개/판매/광고 (대출, 보험, 카드, 증권)
- 투자 자문, 리딩방, 유료 정보 제공
- 소셜/만남/데이팅 플랫폼

---

## 조건부 허용 카테고리

| 카테고리 | 요구사항 |
|----------|----------|
| 의료 | 비치료 원격의료 수익화 금지 |
| 쇼핑 | 고객 서비스, 위조품 방지, 환불 정책 필수 |
| 교육 | 수익 시 국가 자격증 필요 |
| 웹보드 게임 | GRAC 등급, 도박 금지, 확률 공개 의무 |
| 데이팅/소셜 | 19세+, 실명 인증, 가짜 프로필 탐지 |

## 생성형 AI 정책

- AI 사용 사실 **사전 고지** 필수
- 모든 AI 생성 결과물에 **가시적 표시** (라벨, 배지, 워터마크) 필수
- 위반 시 **3,000만원 과태료** 및 서비스 정지

### AI 챗봇/상담 정책
- 위험 키워드 문맥적 필터링
- 자해/불법/타인 위해 요청 거부
- 약물 제조법, 자해 방법 등 전면 금지

---

## 다크패턴 금지
기만적 UX 패턴 사용 금지. 투명성 필수.

---

## 외부 링크 정책
- 별도 앱 설치 유도 **금지**
- 외부 결제/다운로드 페이지 이동 **금지**
- 예외: 법적/규제 링크, 정부/파트너 공식 페이지

---

## 중복 앱 방지
동일 기능 앱을 다른 테마로 반복 출시하여 검색 독점 금지.

---

## 개인정보 보호
- 「개인정보보호법」 제28조 및 「정보통신망법」 제28조 준수
- mTLS 인증서 노출 시 즉시 폐기 후 재발급
- 정책 수시 개정 가능

---

## 민감 콘텐츠 정책
- 3가지 유형: 성적, 폭력적, 범죄 조장
- "경미" ~ "금지" 단계 제한
- 명시적 성적 묘사, 미화된 폭력 = 출시 불가
