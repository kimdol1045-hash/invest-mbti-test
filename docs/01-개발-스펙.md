# 앱인토스(Apps in Toss) 개발 종합 스펙

## 플랫폼 개요

앱인토스는 파트너사가 개발한 서비스를 토스 앱 내부에서 **미니앱(App-in-App)** 형태로 노출하는 슈퍼앱 플랫폼

### 핵심 지표 (2026년 2월 기준)
- 누적 가입자: ~3,000만명
- 누적 미니앱 이용자: 5,100만명+
- 등록 미니앱: 1,000개+
- 파트너사 서비스 지속율: 95%

### 시스템 요구사항
| 플랫폼 | 최소 버전 |
|---------|-----------|
| Android | Android 7 |
| iOS | iOS 16 |
| 사용자 연령 | 19세 이상 |

## 1. 개발 방식

### A) WebView 개발
```bash
npm install @apps-in-toss/web-framework
npx ait init  # granite.config.ts 생성
```
- 지원 프레임워크: React, Vue 3, jQuery
- 빌드 결과물: `.ait` 파일 → CDN 자동 호스팅

**granite.config.ts 예시:**
```typescript
export default defineConfig({
  appName: 'my-app',
  brand: {
    displayName: '%%appName%%',
    primaryColor: '#3182F6',
    icon: 'https://...',
  },
  web: {
    host: 'localhost',
    port: 5173,
    commands: { dev: 'vite', build: 'vite build' },
  },
  permissions: [],
});
```

### B) React Native 개발
```bash
npm create granite-app        # 새 프로젝트
npm install @apps-in-toss/framework  # 기존 프로젝트
```
- 파일 기반 라우팅: `/pages/index.tsx` → `intoss://my-app`
- 모든 라우트는 `intoss://` scheme

### C) 게임 개발
- Unity: WebGL 기반 포팅
- Cocos Creator: v3.8.7+, Node.js 18+

### D) AI 기반 개발 (MCP 연동)
```bash
brew tap toss/tap && brew install ax
claude mcp add --transport stdio apps-in-toss ax mcp start
```

## 2. 인증 & 보안

### 토스 로그인 (필수)
- OAuth 2.0 기반, 자체/제3자 로그인 불가
- 주요 API: `generateOauth2Token`, `refreshOauth2Token`, `loginMe`
- 서버 호출 시 `x-toss-user-key` 헤더 필수

### 토스인증 (Toss Cert)
- `tosscertRequest` - 인증 화면 호출
- `tosscertEncrypt` - 데이터 암호화/복호화

### mTLS 인증서
- 모든 서버 API는 mTLS 필수
- 콘솔에서 인증서 발급, 유효기간 390일
- 엔드포인트: `https://apps-in-toss-api.toss.im/`
- 지원 언어: Kotlin, Python, Node.js, C#, C++

## 3. 결제

### 토스페이 (Toss Pay)
- Server-to-Server (mTLS)
- 결제 실행/승인, 상태 조회, 환불
- 클라이언트 직접 API 호출 금지

### 인앱결제 (IAP)
- 일회성 구매, 구독 (SDK v1.12.0+)
- 상품 카탈로그, 주문 완료, 환불 조회

### 수수료
- 인앱 광고: CBT 기간 15% (현재 0%)
- 인앱결제: 앱스토어 수수료(15-30%) + 토스 5%

## 4. 네이티브 API

| 카테고리 | 기능 |
|----------|------|
| 미디어 | `openCamera`, `fetchAlbumPhotos`, 비디오/오디오 재생 |
| 위치 | 현재 위치 조회, `startUpdateLocation`, 실시간 추적 |
| 연락처 | 디바이스 연락처, 바이럴 마케팅 |
| 공유 | 텍스트/링크 공유 |
| 저장소 | 네이티브 스토리지 (get/set/clear), 클립보드, Base64 파일 저장 |
| 디바이스 | Device ID, 플랫폼 감지, 토스 앱 버전, 네트워크 상태, 로케일 |
| UX | 햅틱 피드백 |
| 광고 | AdMob 전면/보상형/배너 광고 (SDK v1.11.0+) |
